웹 개발 배우면서 쓰면 좋은 VS Code Extension

Extension이란?

쉽게 말해서, 여러분의 삶을 편하게 해줄 VS 코드 확장 프로그램이라고 생각하면 됩니다.

HTML 자동완성, 실시간 뷰어 등 여러 확장 프로그램들이 존재합니다.

여태까지 제가 써보고 괜찮다고 느낀 것들을 추천하겠습니다.

기능

HTML Snippets

HTML 자동완성, 이건 필수 of 필수 입니다.

Python

파이썬을 사용하는데 유용한 기능들을 모두 때려박은 확장 extension입니다. 장고를 쓰실 거면 그냥 필수입니다.

Live Server

여러분들의 HTML을 빨리 확인하기 위한 좋은 extension입니다. 처음에 잘 안되시는 분들이 있던데, 그때는 ctrl + shift + p를 누른다음 open in live server를 치면 나옵니다.

Auto Close Tag / Auto Rename Tag

HTML 태그를 다루는걸 쉽게 해주는 녀석입니다. 위에 녀석은 자동으로 닫는 태그를 만들어주고, 밑에 녀석은 태그를 다른 태그로 바꿔야할때, 여는 태그와 닫는 태그를 동시에 바꿔줍니다. 하나만 다운 받는걸 추천드립니다.

외적 부분

vscode-icons

왼쪽 폴더창에 아이콘들을 좀 더 이쁘게 바꿔줍니다. 각 확장자 마다 어떤 녀석인지 눈에 확확들어오기 때문에 눈이 한결 편해집니다. (이 말고도 아이콘 모양을 바꿔주는 많은 extension이 있습니다. 취향에 맞는걸 검색해서 사용하세요)

Prettier

foo(reallyLongArg(), omgSoManyParameters(), IShouldRefactorThis(), isThereSeriouslyAnotherOne());

이거를

foo(
  reallyLongArg(),
  omgSoManyParameters(),
  IShouldRefactorThis(),
  isThereSeriouslyAnotherOne()
);

이렇게 예쁘게 꾸며줍니다

Rainbow Brackets

bracket을 알록달록하게 만들어 줍니다. 중첩으로 bracket이 쓰여질때 눈에 더 잘보입니다.
